<h3 align="center">What Lazarus does</h3>

<br>

<table align="center">
  <tr>
    <td width="200" align="center"><strong>Governance</strong></td>
    <td>Define what each agent can and cannot do. Set boundaries, permissions, and operational policies across your entire fleet.</td>
  </tr>
  <tr>
    <td align="center"><strong>Memory</strong></td>
    <td>Persistent, structured memory that agents build over time. See what they've learned, correct what they've gotten wrong.</td>
  </tr>
  <tr>
    <td align="center"><strong>Escalation</strong></td>
    <td>When agents hit the edge of their capability, Lazarus routes decisions to humans — with full context, zero lag.</td>
  </tr>
  <tr>
    <td align="center"><strong>Observability</strong></td>
    <td>Real-time visibility into every agent action, decision, and outcome. Audit trails that actually make sense.</td>
  </tr>
</table>
